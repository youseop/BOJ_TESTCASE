저희가 사용하는 테스트 케이스를 이용하시는 것이기 때문에, 문제는 없습니다.

게시판에 올라가있는 반례를 등록하는 것 보다는 직접 유저들이 테스트 케이스를 등록해서 사용하게 하는 것이 좋을 것 같습니다. 반례를 만들어 주는 것도 꽤 시간이 오래 걸리는 일이기 때문입니다.

이런 사이트의 운영은 자유이고, 저한테 허락 받지 않고 운영하셔도 됩니다. 광고도 다셔도 됩니다.

이런 사이트가 예전에도 조금 있었습니다. BOJ 게시판에 홍보 글을 올리면 제가 글을 삭제했던 기억이 있습니다. 제가 다른 사이트 홍보는 막지 않으나, 이런류의 사이트 홍보는 모두 삭제하고 있습니다. 별다른 이유가 있어서 그런 것은 아니고, 게시판 유저의 이동을 막기 위해서 삭제하고 있습니다.

3달 전에 프로그래밍을 시작했는데, 벌써 firebase로 사이트도 만들고, 저는 어렵게 느꼈던 Google, GitHub OAuth 로그인을 추가하신 것을 보니 엄청 대단한 것 같습니다.

참고로 B.O.J가 아니고 BOJ 입니다.

재미있는 프로그래밍 공부 되시길 바랍니다.
